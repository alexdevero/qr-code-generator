# QR Code Generator

A minimal QR code generator app.

### Table of contents

* [Install](#install)
* [Usage](#usage)
* [Support this project](#support-this-project)
* [Code of Conduct](#code-of-conduct)
* [License](#license)

### Install

#### Clone this repo

```bash
git clone https://github.com/alexdevero/qr-code-generator.git
```

#### Install dependencies

```bash
npm install
```

or

```bash
yarn
```

### Usage

#### Run the app

```bash
npm run start
```

or

```bash
yarn run start
```
